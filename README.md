# Halloween Guest List

A fun and interactive web application to manage a Halloween party guest list. Users can add guests, assign them characters/costumes, track likes and dislikes, search, sort, and identify the most liked costumes.

## Features

- **Add Guests:** Enter guest names and characters/costumes.  
- **Edit Guests:** Click on a guest name to edit their details via a modal.  
- **Remove Guests:** Delete individual guests from the list.  
- **Clear Guests:** Clear the entire guest list with one click.  
- **Likes & Dislikes:** Increment likes or dislikes for each guest.  
- **Highlight Top Guest:** The guest with the most likes is highlighted.  
- **Sort Guests:** Alphabetically sort the guest list (A-Z).  
- **Search Guests:** Search guests by name dynamically.  
- **Most Liked Costume:** Displays the costume with the most likes.  
- **Responsive Design:** Works on mobile, tablet, and desktop.  

## Technologies Used

- **HTML5** – Structure of the page.  
- **CSS3** – Custom styling for table headers, highlights, and visual improvements.  
- **JavaScript** – Handles all interactivity (adding, editing, removing, sorting, searching, likes/dislikes).  
- **Bootstrap 4** – Responsive design and styling components.
